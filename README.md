# BlazorAccounting

## Instructions:

- This project is developed using .Net 6. Open the project in VS 2022.
- Run command in Package Manage Console: `update-database` to run DB migrations

## Features:

- Chart of Accounts
- Transactions (Double Entry)
- Dashboard
- Ledger Report
- Trial Balance Report
